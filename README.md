## 🚀 Currently shipping

### [Snake OS](https://snake-os.com) — Retro Nokia Snake meets Solana PvP

A full Solana mainnet PvP wagering game built into a simulated Nokia 3310. 18 in-app screens, real SOL escrow, ORACLE risk scanner, DEGEN pump.fun feed, on-chain market — all wrapped in a retro phone UI.

**Stack:** React 19 · TanStack Start · Cloudflare Workers · Solana web3.js · Supabase  
**Status:** 🟢 Closed beta · token launch targeted Week 3 of the public roadmap  
**Live:** [snake-os.com](https://snake-os.com)  
**Docs:** [docs.snake-os.com](https://docs.snake-os.com) — full feature reference + security model + token plan
